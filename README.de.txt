Verwalte und überwache deinen Boiler über das OpenTherm Gateway und MQTT.

Das OpenTherm Gateway wird zwischen deinem Thermostat und deiner Zentralheizung platziert. Es bietet eine Echtzeitüberwachung von vielen Sensoren und Temperaturen in deinem Heizungssystem. Außerdem ermöglicht es die Steuerung mehrerer erweiterter Einstellungen (VERWENDUNG AUF EIGENE GEFAHR!)

Das OpenTherm Gateway ist ein Open-Source-Gerät. Es kann selbst gebaut oder online gekauft werden (z. B. nodo-shop.nl). Die Firmware, die zur Entwicklung der Homey-App verwendet wurde, wurde von https://github.com/rvdbreemen geschrieben.
