Bestuur en bewaak je CV ketel via OpenTherm Gateway en MQTT.

De OpenTherm Gateway plaats je tussen je thermostaat en je CV ketel. Je krijgt real-time data van vele sensoren en tempeaturen in je verwarmingssysteem. Ook kun je een diverse geavanceerde instellingen doen (GEBRUIK OP EIGEN RISICO!)

De OpenTherm GateWay is een open source apparaat. Je kunt het zelf bouwen, of online kopen (bv nodo-shop.nl). De gebruikte firmware waarmee de Homey app is ontwikkeld is geschreven door https://github.com/rvdbreemen
