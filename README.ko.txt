OpenTherm Gateway와 MQTT를 통해 보일러를 관리하고 모니터링하세요.

OpenTherm Gateway는 당신의 온도 조절기와 중앙 히터 사이에 위치하며, 중앙 난방 시스템의 여러 센서와 온도를 실시간으로 모니터링합니다. 또한 몇 가지 고급 설정을 제어할 수도 있습니다 (사용자 책임 하에 사용!).

OpenTherm GateWay는 오픈 소스 장치입니다. 직접 제작할 수 있거나 온라인에서 구매할 수 있습니다(예: nodo-shop.nl). Homey 앱을 개발하기 위해 사용된 펌웨어는 https://github.com/rvdbreemen에 의해 작성되었습니다.
