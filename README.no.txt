Administrer og overvåk kjelen din via OpenTherm Gateway og MQTT.

OpenTherm Gateway plasseres mellom termostaten din og sentralvarmeren din. Den gir sanntidsovervåking av mange sensorer og temperaturer i ditt sentralvarmesystem. Den lar deg også kontrollere flere avanserte innstillinger (BRUK PÅ EGET ANSVAR!)

OpenTherm GateWay er en åpen kildekode-enhet. Den kan bygges selv, eller kjøpes online (f.eks. nodo-shop.nl). Fastvaren som brukes til å utvikle Homey-appen ble skrevet av https://github.com/rvdbreemen
