Administra y monitorea tu caldera a través de OpenTherm Gateway y MQTT.

El OpenTherm Gateway se coloca entre tu termostato y tu calefactor central. Ofrece monitoreo en tiempo real de muchos sensores y temperaturas en tu sistema de calefacción central. También te permite controlar varias configuraciones avanzadas (¡USAR BAJO SU PROPIO RIESGO!)

El OpenTherm Gateway es un dispositivo de código abierto. Puedes construirlo tú mismo o comprarlo en línea (por ejemplo, nodo-shop.nl). El firmware utilizado para desarrollar la app Homey fue escrito por https://github.com/rvdbreemen
